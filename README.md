# Alphabet Recognition

This project is a deep learning project that recognizes handwritten English alphabets from A to Z using a CNN model.

It supports early literacy learning through an educational handwriting recognition application.

## Features

- Recognizes handwritten English alphabet letters from A to Z
- Uses a Convolutional Neural Network model
- Includes an AlexNet-inspired model experiment
- Trains and evaluates deep learning models
- Visualizes accuracy and loss during training

## Tech Stack

- Python
- TensorFlow / Keras
- CNN
- AlexNet-inspired Model
- NumPy
- Matplotlib
- Jupyter Notebook
